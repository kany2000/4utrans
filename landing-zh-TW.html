立刻開始使用

三步完成安裝，立即體驗智能翻譯

1

克隆或下載項目

獲取 QuickTranslate 最新代碼

git clone https://github.com/kany2000/QuickTranslate.git
2

打開 Chrome 擴展頁面

在位址欄輸入以下位址

chrome://extensions/
3

載入擴展

開啟開發者模式，點擊"載入未封裝項目"，選擇項目資料夾

✓ 開發者模式 → 載入未封裝項目 → 選擇 QuickTranslate

常見問題

遇到問題？這裡有答案

請前往 chrome://extensions/shortcuts 檢查快捷鍵設置，確保沒有與其他擴展衝突。您也可以手動設置新的快捷鍵組合。
請確保選取的區域包含清晰的文字。圖片、特殊字體或背景複雜的文字可能無法準確識別。建議選擇文字對比度高的區域。
目前支援日語、韓語、英語、德語、法語、西班牙語等10+種語言翻譯成中文（簡體/繁體）。系統會自動檢測源語言，無需手動選擇。
點擊擴展圖示打開主介面，在標題欄右側有語言選擇器。支援簡體中文、繁體中文、英語、日語、韓語5種介面語言。設置會自動儲存。
可以嘗試選擇更精確的文字區域，或在高階設置中切換不同的翻譯引擎。Google翻譯適合日常使用，GLM大模型翻譯品質更高但需要API Key。
