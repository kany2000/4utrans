# 📷 截圖翻譯器 (v1.3.0) - 智能網頁翻譯助手

> 🚀 **全新升級** - 基於先進DOM技術的智能翻譯擴展，為您提供無與倫比的網頁翻譯體驗

## 🤝 常見問題

### ❓ **使用問題**
**Q: 快捷鍵不生效？**
A: 前往 `chrome://extensions/shortcuts` 檢查設置，確保無衝突

**Q: 翻譯結果還是原文？**
A: 檢查語言檢測設置，建議使用"自動檢測"

**Q: 無法識別某些文字？**
A: 確保選中區域包含清晰文字，避免圖片或特殊格式

### 🔧 **技術問題**
**Q: 擴展無法啟動？**
A: 重新載入擴展，檢查是否已啟用，必要時重啟瀏覽器

**Q: 在某些網站無法使用？**
A: 部分網站有安全策略限制，請嘗試刷新頁面
