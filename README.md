> 🎉 **选中文字即可快速翻译**，支持 Google、Microsoft、GLM、LLM 多种翻译引擎

**QuickTranslate** 是一款强大的 Chrome 浏览器扩展，采用创新的 DOM 文字提取技术，实现精确的网页文字识别和即时翻译。

## ✨ 核心功能

| 功能 | 说明 |
|:---|:---|
| ⚡ **快捷翻译面板** | 选中文字自动显示翻译按钮，2步完成翻译 |
| 🎯 **精准识别** | DOM 直接提取文字，准确率 98%+ |
| 🌐 **多引擎翻译** | Google / Microsoft / GLM / LLM |
| 🔄 **多引擎对比** | 同时显示三个引擎结果，一目了然 |
| ⌨️ **悬浮翻译** | 按住 Alt 键悬停文字即可翻译 |

## 🚀 快速开始

1. 下载本扩展源码
2. 打开 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹

## ⚙️ 支持的翻译服务

| 服务 | 类型 | API Key |
|:---|:---|:---|
| Google 翻译 | 免费 | ❌ 不需要 |
| Microsoft | 免费 | ✅ 需要 |
| GLM (智谱) | AI 大模型 | ✅ 需要 |
| LLM 自定义 | OpenAI 兼容 | ✅ 需要 |

## ⌨️ 快捷键

| 快捷键 | 功能 |
|:---|:---|
| `Alt + 1` | 智能翻译（自动检测语言） |
