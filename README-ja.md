> 🎉 **テキストを選択するだけで即座に翻訳**、Google、Microsoft、GLM、LLMなど複数の翻訳エンジンに対応

**QuickTranslate** は、革新的なDOMテキスト抽出技術を採用した強力なChromeブラウザ拡張機能で、精密な网页テキスト認識とリアルタイム翻訳を実現します。

## ✨ 主な機能

<details>
<summary>✨ <strong>詳細</strong></summary>

### 🌟 **クイック翻訳パネル（新機能！）**
- **選択して翻訳** - 网页上でテキストを選択すると自動的に翻訳ボタンが表示
- **ワンスクリック翻訳** - 操作ステップが5ステップから2ステップに
- **超高速** - 翻訳速度75%向上、~2秒から~0.5秒
- **精美なパネル** - 紫グラデーンテーマのフローティング翻訳パネル
- **スマート配置** - 選択したテキストの近くに翻訳ボタンが表示
- **ワンスクリックコピー** - 翻訳結果をクリップボードに一瞬でコピー

### 🎯 **精密認識技術**
- **DOM直接抽出** - OCRをスキップ、网页構造から直接テキスト抽出、精度**98%+**
- **スマート領域検出** - マルチポイントサンプリングアルゴリズム
- **自動フィルタリング** - CSSスタイル、コードスニペットを自動除外
- **キャッシュ最適化** - スマートキャッシュメカニズム

### 🧠 **インテリジェント言語処理**
- **AI言語検出** - 深い文字特徴分析ベース、精度**99%+**
- **多言語対応** - 日本語、韓国語、英語、中国語など10+言語に対応
- **コンテキスト理解** - 混合言語コンテンツのインテリジェント認識
- **簡体字/繁体字** - 自動判別

### ⚡ **3つの翻訳モード**
- **クイック翻訳** - テキスト選択 → ボタンクリック → 結果表示（**日常使用におすすめ**）
- **スクリーンショット翻訳** - Alt+1 → 範囲選択 → OCR → 翻訳
- **ホバー翻訳** - Altキー押しながら → テキストにホバー → 自動翻訳
- **マルチエンジン** - Google + マイ_memory バックアップ、成功率**95%+**

### 🎨 **モ踢ンインターフェース**
- **直感的操作** - ドラッグで選択、即时翻訳
- **スムーズアニメーション** - 美しく視覚的なフィードバック
- **スマートヒント** - リアルタイム状態表示
- **レスポンシブデザイン** - 異なる画面サイズに対応

## 🚀 使い方

### 最初の設定
1. **設定を開く** - ブラウザツールバーの📷アイコンをクリック
2. **言語を設定** - 目標言語と認識言語を選択
3. **保存** - **「💾 保存」**ボタンをクリック
4. **使用開始**

### ショートカット（おすすめ）
1. **スマート翻訳** - `Alt+1` を押す
2. **テキストを選択** - ドラッグで範囲を選択
3. **結果を確認** - 翻訳ウィンドウが表示

### クイックパネル（Ctrl+Shift+Q）
1. `Ctrl+Shift+Q` でフローティングパネルを開く
2. テキストを入力して翻訳ボタンをクリック
3. 結果をコピーまたは保存
4. × またはESCで閉じる

### ホバー翻訳（Altキー長押し）
1. 詳細設定で有効化
2. Altキーを押しながらテキストにホバー
3. 翻訳バブルが自動的に表示
4. Altを離して閉じる

## ⚙️ 設定

### メイン設定
- **目標言語** - 翻訳目標言語
- **認識言語** - テキスト認識言語（自動おすすめ）
- **クイック保存** - 設定を即時保存

### 詳細設定
- **翻訳サービス** - Google、Microsoft、GLM、カスタムLLM
- **API設定** - サードパーティサービスのAPIキー
- **クイックパネル** - テキスト選択後に自動表示
- **ホバー翻訳** - Altキーで翻訳（有効化必要）
- **マルチエンジン比較** - Google / Microsoft / LLMの結果を表示
- **自動コピー** - 翻訳結果を自動コピー
- **ショートカット** - カスタムショートカット

### ショートカット

#### デフォルトショートカット
- **Alt+1** - スマート翻訳モード

#### カスタムショートカット
1. 詳細設定 → ショートカットエリア
2. **「変更」**ボタンをクリック
3. 新しいキーの組み合わせを押す
4. 設定は自動保存
5. または `chrome://extensions/shortcuts` で手動設定

## 🔧 よくある質問

**Q: ショートカットが動作しない？**
A: `chrome://extensions/shortcuts` でショートカット設定を確認してください。

**Q: 一部のテキストが認識できない？**
A: 選択した範囲に明確なテキストが含まれていることを確認してください。

**Q: 翻訳結果が元のテキストになる？**
A: 言語検出の問題の場合があります。拡張機能を再読み込みしてください。

**Q: 対応している言語は？**
A: 現在、日本語、韓国語、英語などを中国語に翻訳できます。システムは自動的にソース言語を検出します。
