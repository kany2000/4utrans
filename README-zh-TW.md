> 🎉 **選取文字即可快速翻譯**，支援 Google、Microsoft、GLM、LLM 等多種翻譯引擎

**QuickTranslate** 是一款強大的 Chrome 瀏覽器擴展，採用創新的 DOM 文字提取技術，實現精確的網頁文字識別和即時翻譯。

- **AI語言檢測** - 基於深度字符特徵分析，準確率達 **99%+**

## 🚀 使用方法

### 首次設置
1. **打開設置** - 點擊瀏覽器工具欄中的 📷 圖標
2. **配置語言** - 選擇目標語言（推薦：繁體中文）和識別語言
3. **保存設置** - 點擊 **「💾 保存設置」** 按鈕
4. **開始使用**

### 快捷鍵操作（推薦）
1. **智能翻譯** - 按 `Alt+1` 啟動智能翻譯
2. **選擇文字** - 拖拽框選要翻譯的區域
3. **查看結果** - 自動彈出翻譯窗口

### 快捷面板（Ctrl+Shift+Q）
1. 按 `Ctrl+Shift+Q` 在頁面任意位置彈出浮動面板
2. 在輸入框中輸入文字，點擊翻譯按鈕
3. 結果顯示後可複製或收藏到生詞本
4. 點擊 × 按鈕或按 ESC 關閉

### 懸浮翻譯（按住 Alt 鍵）
1. **啟用功能** - 在高階設置中勾選「啟用懸浮翻譯」
2. **按住 Alt 鍵** - 將滑鼠懸停在任意文字上
3. **查看翻譯** - 自動顯示懸浮翻譯氣泡
4. **鬆開 Alt** - 關閉懸浮翻譯

## ⚙️ 設置選項

### 主界面設置
- **目標語言** - 設置翻譯目標語言（推薦：繁體中文）
- **識別語言** - 設置文字識別語言（推薦：自動檢測）
- **快速保存** - 點擊 **「💾 保存設置」** 立即保存配置

### 高級設置
- **翻譯服務** - Google翻譯、Microsoft翻譯、GLM、LLM自定義
- **API配置** - 第三方服務的API密鑰設置
- **快捷翻譯面板** - 選中網頁文字後自動顯示翻譯按鈕

#### 預設快捷鍵

2. 點擊 **「修改」** 按鈕

## 🔧 常見問題

**Q: 快捷鍵不生效怎麼辦？**
A: 請前往 `chrome://extensions/shortcuts` 檢查快捷鍵設置。

**Q: 為什麼有些文字無法識別？**
A: 請確保選中的區域包含清晰的文字。

**Q: 翻譯結果還是原文怎麼辦？**
A: 這通常是語言檢測問題。請嘗試重新載入擴展。

**Q: 支援哪些語言翻譯？**
A: 目前支援日文、韓文、英文等語言翻譯成中文。系統會自動檢測源語言。
